Project Title

Pizza Perfection

Project Description

Our project is an interactive recipe microsite inspired by Domino’s Pizza’s UI/UX. The website will be a single-page application using only HTML and CSS, where users can interact with animated recipe cards. The site will rely on CSS transforms, transitions, and keyframe animations to create a visually engaging and functional interface without the use of JavaScript. The goal is to combine responsiveness, accessibility, and creativity into a cohesive design that demonstrates mastery of CSS animations.
